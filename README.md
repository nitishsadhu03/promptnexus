## PromptNexus

This is a website where users can share AI-powered prompts that can be used directly in AI chatbots or search engines like chatgpt to generate required results.

## Tech Stacks Used

1. Next js
2. Tailwind CSS
3. React js
4. MonogoDB
5. Next-Auth
